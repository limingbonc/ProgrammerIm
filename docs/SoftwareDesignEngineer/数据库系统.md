## 三级模式-两层映射
> 外模式（用户模式）是**视图级**   
> 概念模式（模式）是**表级**  
> 内模式（存储模式）是**文件级**  

> 为了数据独立性，有了两层映射
>> 外模式 -> 概念模式的映射  
>> 概念模式 -> 外模式的映射
